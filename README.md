# ✨ About The Project

Eunoia Resin Art is a premium full-stack web application designed for preserving memories through handcrafted resin artwork.

The platform allows customers to:

- Order custom resin art pieces
- Preserve wedding bouquets & flowers
- Upload memories and reference images
- Track live order progress
- Pre-book flower preservation kits
- Receive luxury handcrafted experiences

The system also includes a complete admin dashboard for managing:

- Products
- Orders
- Payment tracking
- Preservation timelines
- Pre-booking kit workflows
- Customer submissions

Customers can reserve a preservation starter kit before shipping flowers.

### Kit Includes
- 1kg Silica Gel
- Protective Gloves
- Airtight Container
- Instruction Guide

### Purpose
- Prevent flower discoloration
- Reduce moisture damage
- Preserve petals before courier shipping
